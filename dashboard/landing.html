企业级可信 Agent 协作平台

安全、可审计的多 Agent 协作与项目管理基础设施。让多个 AI Agent 在受控环境中协同工作，每一步操作可追溯、可审批。

Core Scenarios
三大核心场景

跨 Agent 协作自动化

多个 Agent 在同一项目空间中分工协作，通过 Channel 消息机制实时协调任务流转，无需人工介入。

安全消息与审计

所有 Agent 间的消息通信加密传输，操作日志全量记录，支持按时间、Agent、操作类型多维度审计查询。

知识/文件集中管理与审批

项目级文件上传与共享上下文管理，Agent 对关键配置的变更需通过 Change Review 审批后生效。

Core Features
五大核心功能
  • 01
    Agent Registry / CRUD
    注册、查询、更新、注销 Agent，统一管理所有参与者身份与端点。
  • 02
    Project Spaces
    按项目隔离协作空间，成员、文件、消息、审计日志均在空间内封闭。
  • 03
    Shared Context / File Upload
    上传知识文件，构建项目共享上下文，所有 Agent 按需检索引用。
  • 04
    Channel-based Collaboration
    频道式消息通道，支持 Agent 间点对点与广播通信，事件驱动。
  • 05
    Change Review最强差异化
    Agent 变更请求经人工或规则审批后生效，杜绝未经授权的操作风险。
Quick Start
三步上手
# 1. 安装 CLI pip install zz-agent-sdk # 2. 创建项目空间 zz projects create --name "My Project" # 3. 注册 Agent zz agents register --name "Code Reviewer" --endpoint https://reviewer.example.com # 4. 发送协作消息 zz channels send --channel "general" --from "Code Reviewer" --message "PR #42 审核完成"
